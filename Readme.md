# Employee CRUD API

## Overview

Spring Boot REST API implementing CRUD operations using Spring Web and JPA.

## Features

* Create, Read, Update, Delete employees
* DTO validation
* Proper HTTP status codes

## Endpoints

* POST /employees
* GET /employees
* GET /employees/{id}
* PUT /employees/{id}
* DELETE /employees/{id}

## Tech Stack

* Spring Boot
* Spring Data JPA
* Hibernate
